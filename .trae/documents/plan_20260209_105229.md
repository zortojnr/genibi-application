# Genibi Production & Play Store Deployment Plan

## Current State Analysis
Based on my review of the codebase, Genibi has a solid foundation but needs several critical enhancements for production readiness and Play Store compliance.

## Implementation Plan

### **1. Mood Tracker Enhancement**
**Problem**: Current mood tracking uses abstract labels (Positive/Negative/Neutral) and technical terms.
**Solution**: 
- Replace abstract mood direction with human-readable emotions: "Happy", "Sad", "Not sure of my feelings", "Mixed feelings (happy and sad)"
- Update the data model to store these as enums for better analytics
- Ensure the UI reflects these changes in both user and admin views

### **2. Global Disclaimers System**
**Problem**: Missing mandatory disclaimers for mental health apps and Play Store compliance.
**Solution**:
- Create a reusable `DisclaimerBanner` component
- Add disclaimers to key areas:
  - Login/Register screens: "Children under 18 must be supervised"
  - AI Chat: "Genibi is NOT a medical consultation platform"
  - E-Library: "Content from external sources, not medical opinion"
- Ensure disclaimers are visible but non-intrusive (collapsible banners)

### **3. E-Library Admin Upgrade**
**Problem**: Current admin only supports external links, no file uploads or content management.
**Solution**:
- Enhance `AdminLibrary.tsx` with file upload capabilities (PDF, audio, video)
- Add content categorization system (WHO-sourced, verified, general)
- Implement content approval workflow
- Add metadata fields for better organization

### **4. Appointments System Fix**
**Problem**: User appointments may not persist correctly, admin visibility issues.
**Solution**:
- Audit current appointment submission flow
- Ensure proper database writes with error handling
- Verify admin dashboard retrieves all appointments correctly
- Add appointment status tracking and notifications

### **5. Monetization Framework**
**Problem**: No monetization structure for Play Store revenue.
**Solution**:
- Implement user tier system (free/premium) in Firestore
- Gate premium features: Advanced AI chat, full E-Library, priority appointments
- Prepare Google Play Billing integration structure
- Add subscription management UI
